L O R I C S C R I P T Language - C and AT&T x86-64 Assembly (Apple Native)
 January 2023 : Present
LoricScript is a language entirely of my own design meant to become my primary language for future projects. The compiler is built in pure C with nothing beyond the standard libraries and the internal functions are implemented in the target assembly. The compiler is compiled using the Apple toolchain (Clang and LLDB) as opposed to the more common GNU toolchain (GCC and GDB).  August 2022 : December 2022
My most popular tool, JESS, but Less of a Mess, is a color coding VSCode extension meant for Sandia Labs' JESS (Java Expert Systems Shell). JESS is a very niche system, especially in this day and age, and my classmates really struggled with it, so I created the extension to help them. The extension offers both syntax highlighting and documentation hinting for all JESS builtins.

J A V A   C I T Y Educational IDE - Python Pygame
 October 2021 : January 2022
Constructed in my high school's Compilers and Interpreters class to replace an ancient piece of software called Java Town. Java City has a builtin editor for Java and two modes for running it: it can just execute the program or it can go through step by step so that the user can see what is happening, which is especially useful for learning recursion. Java City also provides a visualization of the stack and heap so that new developers can visualize how those are manipulated by their code.
